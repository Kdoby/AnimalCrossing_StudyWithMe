# Study With Dongsoop 🌿

> "당신의 주민들과 자유롭게 공부하세요"

동물의 숲 주민들이 공부하는 영상을 모아, 원하는 주민을 골라 가상의 줌 공부방을 만들어 함께 공부할 수 있는 커뮤니티 기반 스터디윗미 플랫폼.

## 주요 기능

- **줌공부 모드** — 주민 영상을 최대 6개 선택해 줌 그리드 레이아웃으로 루프 재생
- **컨셉영상 모드** — 분위기 있는 장면 영상 1개를 풀스크린으로 루프 재생
- **공부 타이머** — 원하는 시간을 설정하고 카운트다운
- **배경 BGM** — 영상과 별개로 배경 오디오 재생 및 볼륨 조절
- **영상 업로드** — 직접 포착한 주민 영상을 커뮤니티에 공유

## 기술 스택

- React 19 + Vite
- TailwindCSS v4
- React Router v7
- TanStack Query v5
- Axios

## 시작하기

```bash
npm install
npm run dev
```

### 미디어 파일 설정

게임 영상 및 BGM 파일은 저작권 문제로 이 레포지토리에 포함되어 있지 않습니다.  
직접 포착한 영상 클립을 아래 경로에 추가하세요.

```
public/          ← 영상 파일 (.mp4)
public/bgm.mp3   ← 배경 BGM
```
