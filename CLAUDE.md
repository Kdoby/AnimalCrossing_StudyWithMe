# Study With Dongsoop

동물의 숲 주민들이 공부하는 영상을 커뮤니티에서 모아, 원하는 주민들을 골라 가상의 줌 공부방을 만들어 함께 공부할 수 있는 커뮤니티 기반 스터디윗미 플랫폼.

## 프로젝트 개요

커뮤니티 유저들이 Animal Crossing(동물의 숲)에서 주민이 자연스럽게 공부하는 순간을 포착한 영상을 업로드하면, 다른 사용자들이 원하는 주민을 골라 줌 화면 형식으로 함께 공부할 수 있는 웹 서비스.

**슬로건**: "당신의 주민들과 자유롭게 공부하세요"

**디자인 방향**: 감성적이고 포근한 느낌 (Animal Crossing 세계관과 어울리는 따뜻한 파스텔 톤)

## 기술 스택

- **Frontend**: React (Vite), TailwindCSS
- **Backend**: Node.js + Express
- **Database**: MySQL + Sequelize ORM
- **File Storage**: 로컬 스토리지 (추후 S3 등으로 확장 가능)

## 앱 흐름

```
홈(모드 선택)
├── 줌공부 → /zoom (주민 갤러리, 멀티 셀렉트) → /study-room
└── 컨셉영상 → /concept (컨셉 영상 갤러리, 싱글 셀렉트) → /study-room
```

## 핵심 기능

### 1. 홈 화면 (모드 선택)
- 줌공부 / 컨셉영상 두 가지 모드를 카드로 제시
- 모드 선택 후 각 갤러리 페이지로 이동

### 2. 줌공부 갤러리 (`/zoom`)
- 커뮤니티에 업로드된 주민 개인 공부 영상을 카드 형태로 나열 (주민 1명씩)
- 멀티 셀렉트 (최대 6명) → 줌 그리드 레이아웃으로 루프 재생
- 정렬: 최신순, 인기순, 이름순
- 하단 패널에서 공부 시간 선택 후 공부방 진입

### 3. 컨셉영상 갤러리 (`/concept`)
- 여러 주민이 특정 공간·분위기에 함께 잡힌 장면 영상 목록 표시
- 싱글 셀렉트 (1개만 선택) → 풀스크린으로 루프 재생
- 하단 패널에서 공부 시간 선택 후 공부방 진입
- DB: `concept_videos` 테이블 (id, title, videoUrl, thumbnailUrl, createdAt)
- API: `GET /api/concept-videos`

### 4. 공부방 화면 (`/study-room`)
- navigate state로 `{ videos, duration, mode }` 전달받음
  - `mode: 'zoom'` — 멀티 셀렉트, 영상 수 기반 그리드 자동 결정
  - `mode: 'single'` — 단일 영상 풀스크린
- 영상 수 → 그리드: 1개=풀스크린, 2개=1×2, 3~4개=2×2, 5개+=갤러리 뷰
- 각 영상 타일에 주민 이름 네임태그 표시 (줌 스타일)
- 상단 타이머 카운트다운 / 일시정지 / 사운드 토글
- 공부 완료 시 → mode에 따라 `/zoom` 또는 `/concept`로 귀환

### 5. 영상 업로드 (`/upload`)
- 홈 화면에서만 진입 가능. 업로드 유형을 먼저 선택
  - **줌공부**: 주민 1명이 공부하는 순간. 입력: 주민 이름 + 영상(1분 이내) + 썸네일(선택)
  - **컨셉영상**: 여러 주민이 함께 있는 분위기 장면. 입력: 장소/분위기 이름 + 영상(1분 이내) + 썸네일(선택)
- 두 유형 모두 1분 이내 클립을 루프 재생하는 방식으로 동일
